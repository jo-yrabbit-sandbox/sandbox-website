# Sandbox website

A website that displays my PGP public key

## Setup
1. Clone the repository
2. Add PGP key to index.html
3. Deploy to AWS S3

## Things to learn
Features: Text, images
* Display PGP key
* Display images from an image bank in repo
Deployment: AWS
